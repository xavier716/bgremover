# 使用量系统说明

## 📊 当前配置

### 所有用户共享 3 次免费机会
- **总免费次数**: 3 次
- **适用对象**: 所有用户（匿名 + 注册）
- **关键特性**:
  - 匿名用户可以使用这 3 次
  - 注册后**保留已使用的次数**，继续享受剩余免费次数
  - 3 次用完后需要付费购买

### 示例场景

**场景 1：匿名用户用完 3 次**
```
1. 匿名使用: 3 次
2. 剩余次数: 0 次
3. 注册后: 仍剩 0 次（需要购买）
```

**场景 2：匿名使用 2 次后注册**
```
1. 匿名使用: 2 次
2. 剩余次数: 1 次
3. 注册后: 仍剩 1 次（可以继续使用）
4. 用完 1 次后: 需要购买
```

**场景 3：匿名使用 1 次后注册**
```
1. 匿名使用: 1 次
2. 剩余次数: 2 次
3. 注册后: 仍剩 2 次（可以继续使用）
4. 用完 2 次后: 需要购买
```

**场景 4：直接注册（未匿名使用）**
```
1. 注册账号: 0 次使用
2. 免费次数: 3 次
3. 可以使用 3 次后需要购买
```

### 购买选项（3 次用完后）
- **Starter**: $4.99 / 100 次
- **Pro**: $19.99 / 500 次

## 🔧 工作原理

### 1. 共享存储
- 使用 **单一 localStorage 键** 存储所有使用数据
- 键名: `usage_data`
- 匿名和注册用户共享同一个计数器

### 2. 注册时保留数据
- 从匿名用户注册为登录用户时
- **保留所有使用历史**
- **保留剩余免费次数**
- 不会因为注册而重置或丢失数据

### 3. API 路由
- `GET /api/usage/check` - 检查剩余次数
- `POST /api/remove-background` - 使用一次（扣除次数）

### 4. 自动重置
- 免费用户每月自动重置为 3 次
- 付费用户不重置（直到用完购买的次数）

## 🐛 故障排除

### 问题 1: 显示剩余次数，但处理时报错

**原因**: localStorage 和服务器端数据不同步

**解决方案**:
1. 清除浏览器缓存
2. 删除 localStorage 中的 `usage_*` 键
3. 刷新页面重新开始

### 问题 2: 剩余次数不更新

**原因**: 前端未正确更新状态

**解决方案**:
1. 打开浏览器控制台 (F12)
2. 检查 Network 标签
3. 查看 `/api/usage/check` 响应
4. 查看 `/api/remove-background` 响应中的 `remaining` 字段

### 问题 3: 注册后次数重置为 3

**原因**: 这是错误的！注册后应该保留已使用的次数。

**验证**:
- 如果匿名使用了 2 次
- 注册后应该还剩 1 次
- **不应该重置为 3 次**

**如果出现重置问题**:
1. 检查浏览器控制台是否有错误
2. 确认使用的是单一的 `usage_data` 键
3. 查看 localStorage 中是否有多个 usage 键（应该只有 1 个）

## 📝 数据结构

### localStorage 数据格式
```json
{
  "currentUserId": "user_123",           // 当前登录用户 ID（可选）
  "anonymousId": "anon_456",             // 匿名 ID（始终存在）
  "usageCount": 2,                       // 已使用次数
  "totalCredits": 3,                     // 总可用次数
  "planType": "free",                    // 套餐类型
  "lastReset": "2026-03-30T00:00:00.000Z"
}
```

### 关键点
- **单一数据源**: 所有用户（匿名/注册）共享同一个键
- **平滑转换**: 注册时保留所有历史数据
- **匿名 ID 永久保留**: 用于追踪跨会话的使用情况

## 🔄 重置使用数据

### 开发测试用
```javascript
// 在浏览器控制台运行
localStorage.removeItem('usage_data')
location.reload()
```

### 查看当前数据
```javascript
// 在浏览器控制台运行
console.log(JSON.parse(localStorage.getItem('usage_data')))
```

### 查看剩余次数
```javascript
// 在浏览器控制台运行
const data = JSON.parse(localStorage.getItem('usage_data'))
console.log('剩余次数:', data.totalCredits - data.usageCount)
```
